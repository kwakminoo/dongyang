vscode: 가볍고 빠른 실행 / 인텔리센스 / 무한한 확장성 / 통합 터미널 및 git 연동 / 데이터 분석 최적화

정형 데이터: 엑셀의 행과 열이나 관계형 데이터베이스처럼 규칙과 구조를 가진 데이터(엑셀 주소록, 은행 계좌거래 내역 등)

비정형 데이터: 정해진 형태나 구조 없이 자유롭게 생성된 데이터 덩어리(인스타그램 사진, 유튜브 동영상, 카톡 대화, 녹음 등)

데이터 특징: 방대한 양, 속도, 다양성

데이터 분석 6단계:

문제정의 → 데이터 수집 및 정제 → 담색적 데이터 분석 → 데이터 모델링 → 결과 시각화 → 의사결정 및 액션
