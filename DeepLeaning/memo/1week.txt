미분 = 0 = 기울기 = 0

여러 변수중에 특정 변수의 영향수를 알려변 편미분(알고싶은 특정 변수로 식을 나누는 미분법)

경사하강법: 최적화 문제에서 최소값은 기울기 = 0인 지점에서 발생한다.

하지만 처음에 0이 아니면 모델은 조금씩 수정하며 기울기가 0이 되도록 하는 지점으로 이동하는 법. 경사하강법은 최적화의 과정, 또는 방법이다. 

거의 대부분의 최적화 문제에서 답, 정답은 거의 미분(기울기)가 0인 것이다.

오차를 줄이는 방향으로 ㅏㄴ복수정하며 기울기 = 0인 지점(최적화)

LM 트렌스포머

AI = 계산기, 연산처리장치, 규칙기반(사람이 노가다로 다 넣어야됨, 예를들어 100개의 IF문 같은 거임.)

ML = 기계가 학습하는 것, 규칙을 분석하고 규칙을 찾아 새로운 규칙을 스스로 생성하는 것.

은닉층: 데이터를 잘 설명할 수 있는 표현으로 바꾸는 공간.

이를 가능하게 하는 은닉층에 정의된 연산규칙(비선형 함수을 활성화 함수라고 합니다.)

활성화 단계

출발점: 계단함수

시그모이드함수: 계단 함수봐 부르더다, 확률처럼 해석하기 좋다, 로지스틱 회귀와 연결되는 기본 활성화 함수.

출력범위가 0~1일때

변수(X)에따라 0과 1사이를 출력하는데 0과 1에 한없이 가까워질뿐 0과1이 출력되진 않는다.

포화문제: 입력값이 너무 크거나 너무 작으면 기울기가 0에 가까워져 변화에 둔해진다.

변화가 둔해진다는게 무ㅏ임?

출력값이 다양해지지 않는다는 것 가ㅣㅌ음.

학습 속도 저하,: 기울기 소실로 인해 학습이 느려진다.

기울기 소실 = 데이터 소실.

깊은 네트워크 위약: 층이 깊어질수록 정보가 약해지는 기울기 소실 문제 발생.

소프트맥스함수: 여러 클래스 중 하나를 선택할 때.

SIGMOID가 이진분류라면 SOFTMAX는 여러 선택지중 하나를 고르는 확률 분포를 만드는 활성화 함수.

특징: 0과1사이: 모든 출력값은 반드시 0과 1사이의 값을 가진다.

합 = 1: 모든 클래스의 확률을 합산하면 정확히 1이된다.

최종예측: 가장 높은 확률을 가진 클래스가 모델의 최종 예측입니다.

선 그래프 못 그림.

그래서 확률 분포 그래프로 그림.

분류모델.

데이터를 주면 그게 뭔지 분류하는 모델(규칙, 함수)임.

계산원리: 모델의 출력 점수를 지수함수로 변환한 뒤 전체 하으로 나누어 확률 분포를 생성.

적용분야: 이미지분류, 텍스트 감정 분석, 객체인식

-대부분의 분류 딥러닝 모델 출력층은 SOFTMAX를 사용함.
